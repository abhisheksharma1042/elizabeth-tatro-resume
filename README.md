# Elizabeth Tatro - Resume

This repository hosts a styled HTML version of Elizabeth Tatro's professional nanny resume, along with a Markdown version tailored for the Adventure Nannies template.

## Files

- `index.html`: A beautifully styled, modern web resume ready to be hosted via GitHub Pages.
- `resume.md`: The resume content formatted specifically for copy-pasting into the Adventure Nannies template.

## Deployment

This repository is designed to be hosted seamlessly on **GitHub Pages**. Simply enable GitHub Pages in your repository settings to make the `index.html` file live.
